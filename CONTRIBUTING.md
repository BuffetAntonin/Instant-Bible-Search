# Contribuer à instant-bible-search

Merci de vouloir contribuer à **instant-bible-search** !  
Les contributions sont les bienvenues : corrections de bugs, améliorations du code, documentation ou nouvelles fonctionnalités.

## Signaler un bug

Si vous pensez avoir trouvé un bug :

1. Vérifiez d’abord que le problème n’a pas déjà été signalé dans les **Issues**.
2. Si ce n’est pas le cas, ouvrez une nouvelle **Issue**.
3. Décrivez clairement le problème et les étapes pour le reproduire.

Si possible, ajoutez des informations utiles :

- message d’erreur  
- capture d’écran  
- version de **Node.js** ou du navigateur  

## Proposer une amélioration

Si vous souhaitez améliorer le projet :

1. **Forkez** le dépôt.
2. Créez une nouvelle branche pour votre modification.
3. Faites vos changements.
4. Assurez-vous que le projet fonctionne correctement.
5. Ouvrez une **Pull Request** avec une description claire de ce que vous avez modifié.

## Bonnes pratiques

Pour garder un code propre et cohérent :

- respectez le style du projet  
- utilisez **ESLint** pour vérifier le code  
- écrivez des messages de commit clairs  
- évitez les modifications inutiles ou purement cosmétiques  

## Questions

Si vous avez une question ou une suggestion, vous pouvez ouvrir une **Issue** pour en discuter.

Merci pour votre aide et votre intérêt pour le projet 🙌
