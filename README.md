# instant-bible-search

Application de recherche de versets bibliques par mots-clés, construite avec Next.js.

## Prérequis

- Node.js 18+
- npm

## Installation

```bash
git clone https://github.com/AkmaDev/instant-bible-search.git
cd instant-bible-search
npm install
```

## Lancer le projet

```bash
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000) dans ton navigateur.

## Stack technique

- [Next.js](https://nextjs.org) — Framework React
- TypeScript
- ESLint — Linter
