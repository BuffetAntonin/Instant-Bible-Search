# Code de conduite

Ce projet adopte le **Contributor Covenant Code of Conduct**.

Nous nous engageons à maintenir une communauté ouverte, respectueuse et accueillante pour toutes les personnes qui souhaitent contribuer. Chacun est invité à participer dans un esprit de collaboration et de respect.

## Comportements attendus

Les contributeurs sont encouragés à :

- être respectueux et bienveillants dans les échanges  
- accepter les critiques constructives  
- favoriser une atmosphère collaborative et positive  

## Comportements inacceptables

Les comportements suivants ne seront pas tolérés :

- le harcèlement ou les propos discriminatoires  
- les attaques personnelles  
- tout comportement abusif ou irrespectueux  

## Signalement

Si vous êtes témoin ou victime d’un comportement inapproprié, vous pouvez ouvrir une **issue** sur le dépôt ou contacter les mainteneurs du projet.
